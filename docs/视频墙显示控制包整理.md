# 视频墙显示控制包整理：工作站2-1全屏 / 2-1左半屏+2-2右半屏

## 1. 固定信息

```text
协议: TCP
目标端口: 4830
输出节点: 192.168.5.101-192.168.5.150
视频墙: 10列x5行
单屏分辨率: 1920x1080
总画布: 19200x5400
源默认分辨率: 3840x2160
```

## 2. 输入源

```text
工作站2-1组播IP: 224.1.1.55
十六进制: E0 01 01 37

工作站2-2组播IP: 224.1.1.56
十六进制: E0 01 01 38

组播端口: 5000
十六进制: 13 88
```

## 3. 通用发送流程

每次切换均按以下顺序发送：

```text
1. 对50个节点全部发送清屏包 61 FF
2. 对对应节点发送窗口映射包 61 02
3. 对50个节点全部发送提交包 61 B8
4. 对50个节点全部发送刷新包 61 B9
```

## 4. 通用辅助包

这些包发送到所有输出节点：`192.168.5.101-192.168.5.150:4830`。

```text
清屏包:
FB FC 61 FF 00 00 00 00 01 60 FD FE

提交包:
FB FC 61 B8 00 00 00 00 01 19 FD FE

刷新包:
FB FC 61 B9 00 00 00 00 01 1A FD FE
```

## 5. 屏幕IP映射

逻辑屏幕编号：

```text
screen_id = col * 5 + row
ip = 192.168.5.(101 + screen_id)
```

现场补丁：

```text
逻辑屏幕45与46互换
```

即：

```text
if screen_id == 45: screen_id = 46
if screen_id == 46: screen_id = 45
```

## 6. 场景A：大屏全屏显示工作站2-1

### 6.1 说明

工作站2-1画面`3840x2160`铺满整个`10x5`视频墙。

每个输出节点显示一个完整`1920x1080`窗口，但从源画面裁切对应区域。

```text
每列源裁切宽度: 3840 / 10 = 384 = 0x0180
每行源裁切高度: 2160 / 5 = 432 = 0x01B0

src_x = col * 384
src_y = row * 432
src_w = 384
src_h = 432

window_id = 1
mcast_ip = 224.1.1.55
```

### 6.2 包数量

```text
清屏包: 50个
窗口映射包: 50个，全部使用工作站2-1源
提交包: 50个
刷新包: 50个

合计: 200个TCP包
```

## 7. 场景B：左半屏工作站2-1，右半屏工作站2-2

### 7.1 说明

左半屏为第`0-4`列，共`5x5`屏，显示工作站2-1。

右半屏为第`5-9`列，共`5x5`屏，显示工作站2-2。

每个输入源都按`3840x2160`铺满半屏区域，即`5x5`屏。

```text
每列源裁切宽度: 3840 / 5 = 768 = 0x0300
每行源裁切高度: 2160 / 5 = 432 = 0x01B0
src_h = 432
src_w = 768
src_y = row * 432
```

左半屏：

```text
覆盖列: 0,1,2,3,4
源: 工作站2-1
mcast_ip = 224.1.1.55
window_id = 1
src_x = col * 768
```

右半屏：

```text
覆盖列: 5,6,7,8,9
源: 工作站2-2
mcast_ip = 224.1.1.56
window_id = 2
src_x = (col - 5) * 768
```

### 7.2 包数量

```text
清屏包: 50个
工作站2-1窗口映射包: 25个
工作站2-2窗口映射包: 25个
提交包: 50个
刷新包: 50个

合计: 200个TCP包
```

## 8. 可直接给coding agent使用的Python生成器

该脚本会生成完整发送序列，包括目标IP、端口、阶段、十六进制包。

默认只打印JSON；如果需要实际发送，可调用`send_packets()`。

```python
#!/usr/bin/env python3
from __future__ import annotations

import argparse
import json
import socket
import time
from ipaddress import IPv4Address
from typing import Literal


TCP_PORT = 4830

CLEAR_PACKET = "FB FC 61 FF 00 00 00 00 01 60 FD FE"
COMMIT_PACKET = "FB FC 61 B8 00 00 00 00 01 19 FD FE"
REFRESH_PACKET = "FB FC 61 B9 00 00 00 00 01 1A FD FE"

WS21_MCAST = "224.1.1.55"
WS22_MCAST = "224.1.1.56"

WALL_COLS = 10
WALL_ROWS = 5

SCREEN_W = 1920
SCREEN_H = 1080

SOURCE_W = 3840
SOURCE_H = 2160

# 文档字段说明里的稳定值：
# flag_1 = 0009
# flag_2 = 0001
# fixed_window_id = 0001
#
# 注意：附件样例包末尾使用的是 0032 000A 000A。
# 若现场必须完全复现样例行为，可把下面改为:
# FIXED_FLAGS = bytes.fromhex("00 32 00 0A 00 0A")
FIXED_FLAGS = bytes.fromhex("00 09 00 01 00 01")


Mode = Literal["fullscreen-ws21", "split-ws21-ws22"]


def u16(value: int) -> bytes:
    if not 0 <= value <= 0xFFFF:
        raise ValueError(f"u16 out of range: {value}")
    return value.to_bytes(2, "big")


def mcast_bytes(ip: str) -> bytes:
    return bytes(IPv4Address(ip))


def target_ip(col: int, row: int) -> str:
    """
    视频墙逻辑映射:
    screen_id = col * 5 + row
    screen_id 0 -> 192.168.5.101
    screen_id 49 -> 192.168.5.150

    现场补丁:
    逻辑屏幕45与46互换。
    """
    screen_id = col * WALL_ROWS + row

    if screen_id == 45:
        screen_id = 46
    elif screen_id == 46:
        screen_id = 45

    return f"192.168.5.{101 + screen_id}"


def all_target_ips() -> list[str]:
    return [target_ip(col, row) for col in range(WALL_COLS) for row in range(WALL_ROWS)]


def hex_packet(raw: bytes) -> str:
    return raw.hex(" ").upper()


def mapping_packet(
    *,
    mcast_ip: str,
    window_id: int,
    src_x: int,
    src_y: int,
    src_w: int,
    src_h: int,
) -> str:
    """
    生成45字节 61 02 窗口映射包。

    包结构:
    FB FC
    61 02
    00 00 00 21
    window_id_1
    mcast_ip
    mcast_port
    win_x win_y win_w win_h
    window_id_2
    src_x src_y src_w src_h
    flag_1 flag_2 fixed_window_id
    active_cmd
    checksum
    FD FE

    校验和:
    checksum = (sum(packet[2:checksum_pos]) - 1) & 0xFFFF

    这里按附件公式和样例校验结果处理：
    active_cmd 已经追加到packet中，因此会被纳入 sum(packet[2:checksum_pos])。
    """
    packet = bytearray()

    packet += bytes.fromhex("FB FC")
    packet += bytes.fromhex("61 02")
    packet += bytes.fromhex("00 00 00 21")

    packet += u16(window_id)
    packet += mcast_bytes(mcast_ip)
    packet += u16(5000)

    # 输出窗口：每个输出节点本地全屏显示
    packet += u16(0)         # win_x
    packet += u16(0)         # win_y
    packet += u16(SCREEN_W)  # win_w = 1920 = 0780
    packet += u16(SCREEN_H)  # win_h = 1080 = 0438

    packet += u16(window_id)

    # 源裁切区域
    packet += u16(src_x)
    packet += u16(src_y)
    packet += u16(src_w)
    packet += u16(src_h)

    packet += FIXED_FLAGS

    # active_cmd
    packet += bytes.fromhex("01")

    checksum = (sum(packet[2:]) - 1) & 0xFFFF
    packet += u16(checksum)

    packet += bytes.fromhex("FD FE")

    if len(packet) != 45:
        raise ValueError(f"unexpected packet length: {len(packet)}")

    return hex_packet(bytes(packet))


def make_send_item(phase: str, ip: str, packet_hex: str) -> dict:
    return {
        "phase": phase,
        "protocol": "tcp",
        "ip": ip,
        "port": TCP_PORT,
        "packet_hex": packet_hex,
    }


def build_fullscreen_ws21_mappings() -> list[dict]:
    """
    场景A：
    大屏全屏显示工作站2-1。
    """
    items: list[dict] = []

    src_w = SOURCE_W // WALL_COLS  # 384
    src_h = SOURCE_H // WALL_ROWS  # 432

    for col in range(WALL_COLS):
        for row in range(WALL_ROWS):
            packet = mapping_packet(
                mcast_ip=WS21_MCAST,
                window_id=1,
                src_x=col * src_w,
                src_y=row * src_h,
                src_w=src_w,
                src_h=src_h,
            )
            items.append(make_send_item("mapping_ws21_fullscreen", target_ip(col, row), packet))

    return items


def build_split_ws21_ws22_mappings() -> list[dict]:
    """
    场景B：
    左半屏显示工作站2-1，右半屏显示工作站2-2。
    """
    items: list[dict] = []

    half_cols = WALL_COLS // 2  # 5
    src_w = SOURCE_W // half_cols  # 768
    src_h = SOURCE_H // WALL_ROWS  # 432

    for col in range(WALL_COLS):
        for row in range(WALL_ROWS):
            if col < half_cols:
                # 左半屏：工作站2-1
                packet = mapping_packet(
                    mcast_ip=WS21_MCAST,
                    window_id=1,
                    src_x=col * src_w,
                    src_y=row * src_h,
                    src_w=src_w,
                    src_h=src_h,
                )
                phase = "mapping_ws21_left_half"
            else:
                # 右半屏：工作站2-2
                packet = mapping_packet(
                    mcast_ip=WS22_MCAST,
                    window_id=2,
                    src_x=(col - half_cols) * src_w,
                    src_y=row * src_h,
                    src_w=src_w,
                    src_h=src_h,
                )
                phase = "mapping_ws22_right_half"

            items.append(make_send_item(phase, target_ip(col, row), packet))

    return items


def build_sequence(mode: Mode) -> list[dict]:
    """
    返回完整发送序列：
    1. 50个清屏包
    2. 50个窗口映射包
    3. 50个提交包
    4. 50个刷新包
    """
    sequence: list[dict] = []

    for ip in all_target_ips():
        sequence.append(make_send_item("clear", ip, CLEAR_PACKET))

    if mode == "fullscreen-ws21":
        sequence.extend(build_fullscreen_ws21_mappings())
    elif mode == "split-ws21-ws22":
        sequence.extend(build_split_ws21_ws22_mappings())
    else:
        raise ValueError(f"unknown mode: {mode}")

    for ip in all_target_ips():
        sequence.append(make_send_item("commit", ip, COMMIT_PACKET))

    for ip in all_target_ips():
        sequence.append(make_send_item("refresh", ip, REFRESH_PACKET))

    return sequence


def send_tcp_packet(ip: str, port: int, packet_hex: str, timeout: float = 1.0) -> None:
    raw = bytes.fromhex(packet_hex)
    with socket.create_connection((ip, port), timeout=timeout) as sock:
        sock.sendall(raw)


def send_packets(sequence: list[dict], *, interval_ms: int = 0, timeout: float = 1.0) -> None:
    """
    实际发送函数。
    coding agent需要真实下发时调用此函数。
    """
    delay = interval_ms / 1000

    for item in sequence:
        send_tcp_packet(item["ip"], item["port"], item["packet_hex"], timeout=timeout)
        if delay > 0:
            time.sleep(delay)


def main() -> None:
    parser = argparse.ArgumentParser()
    parser.add_argument(
        "mode",
        choices=["fullscreen-ws21", "split-ws21-ws22"],
        help="fullscreen-ws21: 全屏显示工作站2-1；split-ws21-ws22: 左2-1右2-2",
    )
    parser.add_argument(
        "--send",
        action="store_true",
        help="实际通过TCP发送；不加则只打印JSON",
    )
    parser.add_argument(
        "--interval-ms",
        type=int,
        default=0,
        help="每个TCP包之间的间隔毫秒数",
    )
    parser.add_argument(
        "--timeout",
        type=float,
        default=1.0,
        help="TCP连接超时时间，单位秒",
    )

    args = parser.parse_args()

    sequence = build_sequence(args.mode)

    if args.send:
        send_packets(sequence, interval_ms=args.interval_ms, timeout=args.timeout)
    else:
        print(json.dumps(sequence, ensure_ascii=False, indent=2))


if __name__ == "__main__":
    main()
```

## 9. 使用示例

只生成包，不发送：

```bash
python video_wall_packets.py fullscreen-ws21 > fullscreen_ws21_packets.json
python video_wall_packets.py split-ws21-ws22 > split_ws21_ws22_packets.json
```

实际发送：

```bash
python video_wall_packets.py fullscreen-ws21 --send
python video_wall_packets.py split-ws21-ws22 --send
```

如果现场设备需要包间隔，使用：

```bash
python video_wall_packets.py fullscreen-ws21 --send --interval-ms 10
python video_wall_packets.py split-ws21-ws22 --send --interval-ms 10
```

## 10. coding agent实现要点

```text
1. 不要直接向一个广播地址发包；按目标节点IP逐个TCP连接发送。
2. 每个包发送到端口4830。
3. 每次切换前必须先发61 FF清屏。
4. 窗口映射包61 02必须按每块屏单独生成，不能一包覆盖整墙。
5. 最后必须对全部50个节点发送61 B8和61 B9。
6. 逻辑屏幕45和46必须互换，否则最后一列顶部两块屏会错位。
7. 工作站2-1组播IP固定为224.1.1.55，即E0 01 01 37。
8. 工作站2-2组播IP固定为224.1.1.56，即E0 01 01 38。
```
